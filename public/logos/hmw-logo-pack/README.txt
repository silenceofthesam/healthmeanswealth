FORMAT GUIDANCE
---------------
  SVG — use for websites, digital documents, anything that scales (recommended)
  PNG — use for platforms that don't accept SVG (Word docs, Facebook, email clients)

  For professional print (banners, shirts, certificates): give the printer the SVG file.
  SVG is vector — it will print sharp at any size, including very large format.

  For embroidery: give the embroiderer the black icon SVG (hmw-icon-black.svg) 
  and specify the two thread colours: Forest #2D5016 and Sprout #A8D070.

  For rubber stamps: use hmw-logo-horizontal-black.svg or hmw-logo-stacked-black.svg.
